# My Conference Radar

脳機能・神経放射線・MRI関連の学会について、開催日程と演題締切（確定/予想）を集約しています。  
締切の予想は前年度実績などをもとに作成していますが、情勢によって大幅に変わる可能性があるためご注意ください。  
更新は不定期です。日程が確定した場合や、掲載してほしい学会がある場合は、Issueをご作成ください。

- 一覧：Conferences
- 締切表示：
  - `deadline.text`：表示用（例：10月、10月上旬、4–5月、未発表）
  - `deadline.sort`：並べ替え・期限切れ判定用（ISO形式 `YYYY-MM-DD`。不明な場合は未入力で構いません）
- 期限切れの締切は「Show past deadlines」で表示/非表示を切り替えられます。

注意：締切は主催側の告知で変更されることがあります。最終的には各学会の公式ページでご確認ください。
