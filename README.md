# conf_rader

Collection of upcoming abstract deadlines for conferences related to brain function, neuroradiology, and MRI that I am interested in. This list focuses primarily on Japanese conferences, with some international conferences included. Dates are rough estimates, so please be sure to check the official websites.

私が関心のある、脳機能・神経放射線・MRI関連の学会の今後の演題締切をまとめた一覧です。日本の学会を中心に、一部国際学会も含みます。日付は概算の予想ですので、必ず公式サイトをご確認ください。
